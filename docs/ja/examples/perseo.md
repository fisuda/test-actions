## Perseo の健全性チェック

Perseo が起動したら、次のコマンドでステータスを確認できます:

#### リクエスト:

```bash
ngsi version --host perseo.local --pretty
```

#### レスポンス:

```json
{
  "error": null,
  "data": {
    "name": "perseo",
    "description": "IOT CEP front End",
    "version": "1.27.0"
  }
}
```
